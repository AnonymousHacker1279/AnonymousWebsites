How to use the Raspberry Pi as a FM Transmitter

Estimated Time to Make: ~10 Minutes

Description: This project will allow you to broadcast your own music on FM radio up to ~200 feet. NOTE: See notice at bottom of page.

  1. Prepare your Pi

    Make sure you have Raspbian installed on your Pi for this project. You can download it from here. Next, get a piece of wire (about 20 cm in length) and attatch it to pin 4 of the GPIO bus. See the image below for a diagram.


  2. Install Software

    At the command line, type git clone https://github.com/markondej/fm_transmitter to download the transmitter files. Next, change directories into the fm_transmitter folder. Type sudo make. And finally, type sudo apt-get install libsox-fmt-all.


  3. Run the Software

    Assuming that you already have a .wav audio file in the fm_transmitter folder, type sudo ./fm_transmitter -f 89.3 -r AUDIO.WAV to start broadcasting music. You can replace the "89.3" with any FM band. Now you can use a FM radio to tune into your music!


NOTICE:

This type of radio can interfere with emergency equipment. For example, if you live near an airport, the radio waves can cause audio transmission to the pilot to become distorted. If you couldn't guess, that is a MAJOR offense. It's usually safest to use this away from cities and airports, with a small antenna, for a short amount of time.

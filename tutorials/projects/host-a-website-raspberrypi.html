How to Host a Website using Raspberry Pi

Estimated Time to Make: ~30

Description: This project will allow you to host HTML-encoded pages online.


Have you ever wanted to make a website? Or do you know how, but struggle to get it online? Want to do it for free? Well, by using the Raspberry Pi, you can! You just need to know how to program in HTML, the "internet's language".


  1. Prepare your Raspberry Pi:

    You will need to have Raspbian installed on your device for this to work. You can download the image here.After you have installed Raspbian (or if you already had it installed), open Terminal and type sudo apt-get update && sudo apt-get upgrade to update the Pi.


    2. Sometimes, a reboot is required. If the terminal says you should reboot, use sudo reboot to reboot the device.


  2. Install Apache:

    At the terminal, type sudo apt-get install apache2 to install the Apache web server. Once the install is complete, open a web browser on another device and type your Pi's IP address. If you don't know the IP address, type ifconfig and network information will appear.


  3. Add your own HTML files:

    You probably don't want the default page as your website. You can use your own file in place of the default. Go to "/var/www/html" and delete the file named "index.html". Replace this with your own file, naming it "index.html". Try to go back to your website, and your page will appear!


  4. Make your website available from outside your network:

    Due to the way web servers work, you won't be able to access your website anywhere except from within your own network. However, you can use port forwarding to bypass this restriction. Log onto your router's configuration page and select the port forwarding option. You need to forward port 80 and port 443 (HTTP and SSL). After you have completed this, type the Pi's GLOBAL IP address into a browser. This can be found by typing "What's my IP" into a web search.


  5. Use a dynaminc DNS server instead of an IP address:

    You probably don't want to make people remember a set of numbers to access your website. No other website only has access through an IP address. However, you can use a free website called "noip.org". This website lets you use a dynamic DNS name to route to your IP. The end result is people using a website URL instead of an IP address. Go to "noip.org" and create a free account. Follow the steps on their website to make a DNS name.


    6. Now that your website is completed, share your URL with others and expand your site with more pages!

Get Started with HTML

Description: Learn the language of the internet

HyperText Markup Language, or HTML, builds almost every website in existance. It's an easy to learn language and is very popular. In this tutorial, you will learn how to create your first HTML script.

  1. Get a text editor:

    Usually, an application for text editing will be included with your OS, such as Notepad. These editors will work fine. However, you can use more advanced editors such as Notepad++ and Visual Studio to make it easier to work. For this tutorial, we will use Notepad++.

    Take a moment to compare what you wrote and what you see in the web browser. Notice how the <title> tag reads the same as the tab title, and how the <h1> and <p> tags read the same as the header and paragraph. See what we mean when we say HTML is easy to learn? <title> tags represent page titles, <h1> tags represent headers, and <p> tags represent paragraphs. You can choose the header size by using a different number (only 1-6 are supported).

    Try to edit the code using what you know to get the result above.



    Use <h1> through <h6> tags to create headers of different sizes.

    Now that you know what those tags do, you are probably wondering what the other tags do. Simple enough, the <!DOCTYPE HTML> tag tells the computer that the document is written in HTML. The <head> tag is content you don't see on the screen, such as style sheets or JavaScript code. Finally, the <body> tag contains content that you see, like the headers and paragraphs.

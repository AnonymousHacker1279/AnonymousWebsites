Our Timeline

November 14, 2018

Started creating website pages in simple HTML. It looked like it was made in the 90s.

November 21, 2018

Added CSS and other designs to website. At this point, there were only three pages!

December 5, 2018

Added Experimental Pages section and fixed numerous bugs. Thanks to all who reported them!

January 9, 2019

Happy New Year! We added games to our website to celebrate!

March 20, 2019

Added front page animations and meme loader, from suggestions.

May 15, 2019

Complete website shutdown. The internet service provider for the website hosting device blocked all inbound ports on device to reduce bandwidth.

June 1, 2019

Website revival. After backing up all website data, we wiped our host device and used a newer operating system installation. After replacing the files and installing our host, it worked! However, we did change the end of our URL from ".hopto.org" to ".ddns.net".

June 3, 2019

Created local proxy servers. The website currently uses proxy server 2, however, all versions can be found in the Experimental Pages section of the website.

July 10, 2019

Now using HTTPS. This ensures data sent is secure, and you can see the green padlock in the address bar.
